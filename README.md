HelloMiss
=========

HelloMiss is a little application wich allows you to browse and save images on :
* [Bonjour Madame][1]
* [Bonjour Mademoiselle][2]
* [Bonjour La Bombe][7]
* [Bonjour ma belle][8]

Dependencies
------------
* [jsoup][3]
* [SlidingMenu][4]
* [ActionBarSherlock][5]
